# Blog

Personal Hugo blog powered by the **hugo-bearblog** theme.

## Quick start

1. Install Hugo (extended).
2. Run the dev server:
   ```
   hugo server -D
   ```
3. Build the site:
   ```
   hugo
   ```

## Content layout

- Homepage: `content/_index.md`
- Posts: `content/blog/`
- Theme: `themes/hugo-bearblog/`

The homepage lists the most recent posts from the `blog` section.
